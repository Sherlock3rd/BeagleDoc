## 1. 设计理念 (Design Philosophy)

基于 "用户至上，生态驱动" 的核心哲学，PVP 玩法不应成为阻碍非 PVP 玩家体验的壁垒，而应成为一种**可选择的进阶体验**。

本方案的核心逻辑是 **"渐进 (Progression)"**：
1.  **整体渐进 (Overall Progression)**: 不同的 PVP 地块承担不同的 **风险-回报 (Risk-Reward)** 等级。
2.  **地图渐进 (Map Progression)**: 通过地理位置的差异化投放，自然筛选目标用户，避免"强制PVP"。

## 2. 地块分级体系 (Tile Leveling System)

在 7日版本中，我们将构建两级 PVP 地块体系，以满足不同层级玩家的需求。

### 🛡️ Level 1: 狩猎旷野 (The Safari Wilds) - 特殊比宝捕捉
> **"这里有外面见不到的稀有比宝，前提是你能活着把它们带出来。"**

*   **定位**: 泛用户入口，非安全区，核心驱动力是**捕捉特殊比宝**。
*   **开启规则**: **常驻 (Always On)**。
    *   **全天候**: 没有集中时段限制，随时进入，随时开始。
    *   **刷新机制**: 资源点和野怪按照固定频率独立刷新，不依赖特定活动。
*   **核心目标 (Priority)**: 提供一个**非安全区**，让玩家为了捕捉稀有资源（特殊比宝）而承担适度风险。
*   **风险等级**: **Low Risk** (无掉落 / 仅耐久损耗 / 快速复活)。
*   **玩法收益**:
    *   **PVP 收益**: 击败竞争对手，确保捕猎环境。
    *   **核心收益 (Special Beagle)**: **"特殊比宝捕捉"**。
        *   **独占生态**: 该区域刷新地图其他地方没有的**特殊比宝 (Special Beagles)**（如异色、特殊词条、限定亚种）。这是驱动 PVX 玩家进入的核心动力。
        *   **隔离型机制**: 依然保留相对独立的采集/捕捉点，但高价值目标往往伴随着争夺。
    *   **日活接入 (DAU System)**: 将"前往旷野捕捉/观察"列为每日活跃任务。
*   **社交规则**: **联盟为单位 (Alliance Based)**。
    *   延续大地图的联盟社交关系。
    *   同联盟成员互为友好，非联盟成员可攻击。
    *   允许联盟组织"清场"或"包场"行为。
*   **地图落位原则**: **傍路而不挡路 (Near, not On)**。
    *   **口袋式分布**: L1 地块应设计为 **"口袋区" (Pocket Zones)**，吸附在主干道两侧。

### ⚔️ Level 2: 幻装派对 (The Masquerade) - 外观争夺
> **"今晚的主角只能有一个。穿上最闪亮的衣服，去抢那个限定头套！"**

*   **定位**: 进阶 PVP，强调**趣味性**与**外观展示**。
*   **基础状态 (Inheritance)**: **完全继承 L1 玩法**。
    *   **非活动期间**: 该地块的表现与 **Level 1 狩猎旷野** 完全一致（刷新特殊比宝、无额外掉落）。玩家可以正常进行捕捉和采集。
*   **活动开启规则**:
    *   **时间窗口**: 每日 **UTC 11:00 ~ 13:00** 之间随机触发。
    *   **预告广播**: 活动开始前 **5分钟**，播放全服广播告知具体坐标/区域。
*   **核心玩法 (Event Mechanic)**: **幻装宝箱争夺**。
    *   **随机生成**: 在该区块内的**随机坐标**刷新一个【幻装宝箱】。
    *   **全图可见**: 宝箱位置在当前区域的**小地图 (Mini-map)** 上对所有玩家可见（高亮图标）。
    *   **争夺逻辑**: 玩家需前往并开启宝箱（可能需要读条或持有一定时间），成功开启者获得限时外观奖励。
*   **设计目标**: 
    *   **趣味争夺**: 弱化"杀戮"感，强化"争抢"感（如抢夺话筒、抢占舞台中心、抢夺空投礼盒）。
    *   **聚拢时段**: 强调**限时开启**，打造全服瞩目的"狂欢时刻"。
*   **风险等级**: **Mid Risk** (战败可能丢失本次获得的活动积分/道具，但不掉落自身装备)。
*   **玩法收益**:
    *   **非数值奖励**: 绝对**不产出**核心战斗数值（避免逼肝）。
    *   **限时外观 (Cosmetics)**: 提供**全服唯一/限时**的独特外观（如发光头套、变身道具、拖尾特效）。主打"我最闪耀"的炫耀心理。
*   **社交规则**: **联盟为单位 (Alliance Based)**。
    *   延续联盟社交，鼓励联盟层面的争夺和对抗（如抢夺 Boss 或占领点）。
*   **地图落位原则**: **深巷分支 (Deep Branch)**。
    *   **分支地块**: L2 地块放置在地图的**末端分支**或**独立区域**。

## 3. 地图拓扑结构示意 (Map Topology)

```mermaid
graph TD
    subgraph Safe_Zone [安全/中立区域]
        Town[城镇中心]
        MainRoad[== 主干道 (Main Road) ==]
        SafeWild[野外安全采集区]
    end

    subgraph PVP_L1 [Level 1: 狩猎旷野]
        L1_Node1((L1 捕猎点 A))
        L1_Node2((L1 捕猎点 B))
    end

    subgraph PVP_L2 [Level 2: 幻装派对]
        L2_Node1{{L2 核心舞台区}}
    end

    %% 连接关系
    Town --> MainRoad
    MainRoad --> SafeWild
    
    %% L1 依附于主路，但不阻断
    MainRoad -.->|主动拐入| L1_Node1
    MainRoad -.->|主动拐入| L1_Node2
    
    %% L2 位于深处分支
    SafeWild -->|长途跋涉| L2_Node1
    L1_Node1 -.->|进阶路径| L2_Node1

    style MainRoad fill:#e1f5fe,stroke:#01579b,stroke-width:4px
    style L1_Node1 fill:#fff9c4,stroke:#fbc02d,stroke-width:2px
    style L1_Node2 fill:#fff9c4,stroke:#fbc02d,stroke-width:2px
    style L2_Node1 fill:#ffcdd2,stroke:#c62828,stroke-width:2px,stroke-dasharray: 5 5
```

### 设计解读
1.  **主干道 (Main Road)**: 是一条贯穿地图的宽阔通道，保证任何玩家（即使是 0 战力）也可以安全地从城镇移动到远方。
2.  **L1 口袋**: 像高速公路的"服务区"或"匝道"，依附主路存在。热闹，诱人，但你不进去也没事。
3.  **L2 分支**: 像森林深处的"遗迹"，你需要专门离开大路，穿过野外才能到达。

## 4. 7日版本详细执行方案 (Execution Plan)

### 4.1 核心体验目标
*   **验证分层**: 确认玩家能理解 L1 和 L2 的区别（通过 UI 提示、地块颜色、进入警告）。
*   **流量控制**: 确保主干道畅通，同时 L1 区域有足够的人气（通过奖励引导）。

### 4.2 具体地块配置

| 属性 | Level 1: 狩猎旷野 (Safari) | Level 2: 幻装派对 (Masquerade) |
| :--- | :--- | :--- |
| **地块标识** | 黄色爪印标识 (Beagle Zone) | 平时: 同L1 / 活动时: 红色面具标识 |
| **进入警告** | 弱提示 ("即将进入非安全捕猎区") | 活动期间: **强弹窗** ("限时活动正在进行") |
| **战斗机制** | 击败倒地，快速复活，无损失 | **平时同L1**。活动期间可能增加复活冷却 |
| **核心产出** | **隔离型比宝 (Beagle)**、荣誉、日活点数 | **平时同L1** + **活动宝箱 (限时外观)** |
| **刷新规则** | **常驻 (Always On)**，随时体验 | **UTC 11:00-13:00** 随机触发，前5分钟广播 |
| **社交模式** | 联盟对抗 + 散人混战 | 联盟有组织的对抗 (抢夺宝箱) |

### 4.3 待办事项 (To-Dos)
- [ ] **机制实现**: 
    - [ ] L1: 实现"隔离型比宝"获取逻辑（如安全采集点）。
    - [ ] L1: 接入日活系统（Daily Task）。
    - [ ] L2: 实现限时开启与全服广播机制。
- [ ] **地图编辑**: 规划 L1 口袋区与 L2 分支区。
- [ ] **奖励配置**: 配置 L2 的限时外观奖励池。
